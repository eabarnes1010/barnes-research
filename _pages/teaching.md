## Colorado State University

**ATS 780-A8 — Weather-to-Climate Data Driven Forecasting** _(Fall 2024)_
Data-driven approaches (e.g. machine learning) for forecasting are proving to be incredibly powerful — transforming prediction science.

**ATS 655 — Objective Analysis for the Atmospheric Sciences** _(Spring, annually 2014–2025)_
Overview of the methods used to interpret data sets in the atmospheric and oceanic sciences, including basic statistics, matrix methods, spectral filtering & time series analysis. Typed notes available.

**ATS 780A7 — Machine Learning for the Atmospheric Sciences** _(Spring 2022, Fall 2023)_
Overview of machine learning methods used to interpret data sets in the atmospheric/Earth sciences, with emphasis on implementation and interpretation.

**ATS 601 — Atmospheric Dynamics I** _(Fall 2016–2021)_
Basics of atmospheric dynamics and geophysical fluid dynamics on a rotating sphere. [Typed notes](https://drive.google.com/file/d/1WJhF3kz0QEkaSE1q5Red7y5DxJb07zdV/view).
